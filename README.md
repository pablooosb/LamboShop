<h1 align="center">Installation steps</h1>
<p align="center">The first step is to clone the repository. To do this, navigate to the folder where you want to clone it, open the console, and type <code>git clone https://github.com/pablooosb/LamboShop.git</code>. It's crucial to do this correctly, otherwise you won't be able to view the videos, as we're using Git LFS (Git Large File Storage).</p>
<p align="center">For this to get working, we need to install in first place the NPM (Node Package Manager). To do this, go to the "front" folder and there open the terminal to type <code>npm install</code>. Once installed, you should see a see a folder called "node_modules".</p>
<p align="center">Then go to the parent folder. There, you will need to install Git LFS (Git Large File Storage). To do this, open the terminal and type <code>git lfs install</code>, and once it finishes, type <code>git lfs pull</code>. With this, the proyect should work without problems.</p>
